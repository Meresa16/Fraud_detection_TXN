# Bank Transactions Fraud Detection

## Project Overview
Detect fraudulent bank transactions for Mobile App and USSD channels. Includes real-time fraud alerts.

## Dataset
- Source: Kaggle "Bank Transactions Fraud Detection"
- Features: transaction amount, hour, channel, customer average, device, location

## Methodology
1. EDA and Visualization
2. Feature Engineering (basic + advanced)
3. Rule-Based Fraud Detection
4. Machine Learning Models (Random Forest, Logistic Regression, XGBoost)
5. Real-Time Fraud Alert Simulation
6. Interactive Dashboard with Streamlit

## Results
- Classification reports for multiple ML models
- Fraud analysis by channel and hour

## Future Work
- Deep learning / anomaly detection
- Integration with real banking systems
- Advanced dashboards
